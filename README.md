# Jersey Request Catalog

Custom English-only web shop request catalog for sports-inspired jerseys and fanwear.

## Goal

Build a Next.js + TypeScript + MySQL + Prisma + Cloudflare R2 + Railway project with:

- Public catalog website
- Custom admin CMS
- Product and category management
- Customer request CRM
- SEO controls
- Google Search Console / Analytics / Tag Manager script inputs
- Cloudflare R2 image uploads
- WhatsApp and Telegram contact settings
- AI Jersey Fit Assistant

## Business Model

This project should work as a request catalog, not a direct official branded merchandise shop. Customers browse styles, choose size and fit, then submit a request. The manager confirms availability, delivery time, specifications and final price before payment.

## Important Language Rule

Use neutral public wording such as:

- sports-inspired jersey
- basketball style
- football style
- baseball style
- hockey style
- fanwear style
- custom request

Avoid public claims like:

- official
- licensed
- authentic
- approved by league/team/brand
